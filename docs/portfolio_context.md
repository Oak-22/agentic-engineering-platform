# Portfolio Context

This repository is part of an AI-native engineering methodology layer.

Its reusable template surface is the AI Agent Instruction Control Plane:
a repo-portable scaffold for layered agent instructions, runtime
adapters, provenance logs, selective context loading, and auditable task
execution.

In portfolio use, this repository supports project-facing repositories
by documenting reusable collaboration patterns, agent workflows, and
durable knowledge-capture practices.

Project-facing repositories include:

- `myHealth`
- `digital_asset_processing_pipeline`
- `ai-infrastructure-financial-warehouse`

The portfolio role should remain separate from the template's portable
developer-facing contract. Personal instructions, local symlink targets,
and project-specific operating history belong outside the reusable
scaffold unless they have been generalized into public-safe examples.
