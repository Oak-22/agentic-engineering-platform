## What This Tests

This path tests whether prose instructions alone are enough to produce
a strong v2 Mermaid source while preserving the existing semantic
architecture.

It does not provide visual evidence of manual Excalidraw edits, so it
should reveal what the skill loses when it cannot inspect human layout
refinements.
