# AI Knowledge Promotion Diagram

This folder contains the AI knowledge promotion / workflow-derived
learning platform diagram family.

The diagram represents the platform layer that turns personal, team,
and organizational engineering knowledge into durable control-plane
artifacts, learning retrieval signals, and feedforward/feedback
lifecycle inputs.

## Folders

- `predecessor/`
  Historical Draw.io source for the broader personal/team/org knowledge
  capture and integration concept. This predates the Excalidraw-based
  workflow and is kept as provenance.
- `baseline-v1/`
  Current v1 Mermaid source, Excalidraw scene, and exported image.
- `briefs/`
  Concrete and reusable update briefs for diagram iteration.
- `experiment/`
  A1/A2/A3/B experimental matrix for evaluating Mermaid-first and
  Excalidraw-native update workflows.
- `promoted/`
  Reserved for the selected final diagram after comparison.

## Concept Scope

The predecessor diagram is broader than the workflow-derived retrieval
system alone. It represents personal, team, and organizational knowledge
capture and integration.

`docs/periodic_learning_retrieval_system.md` describes one subcomponent:
the learning retrieval layer that can be built on top of the broader
knowledge promotion platform.
